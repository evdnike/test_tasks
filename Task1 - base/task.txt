Задача:

Написать функцию, которая будет принимать на вход лог for_test.log и выводить последние изменения в параметрах eid:

Алгоритм действий:
    - найти в логе две последние записи eid
    - вывести в консоль две строки "Предпоследние eid: " и "Последние eid: "
    - найти отличия в ключах и значениях в последнем и предпоследнем словарях eid
    - вернуть в return словарь с найденными отличиями.

Дополнительные требования:
    - для открытия файла использовать контекстные менеджер with
    - использовать slicing для перебора списков в обратном направлении
    - Использовать словари для хранения параметров eid, например строка параметров “PDO.1;ant.2;bda.1;” должна быть представлена в виде {'PDO': '1', 'ant': '2', 'bda': '1'}
