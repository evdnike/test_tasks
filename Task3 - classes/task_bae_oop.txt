Задача:

Первый класс
Атрибуты:
    количество выигранных игр
    количество ничейных игр
    количество проигранных игр
    количество забитых голов
    количество пропущенных голов


Атрибуты должны инициализироваться при создании объекта класса.


Методы:
    добавить результат матча в формате [количество забитых голов], [количество пропущенных голов].
    подсчет количества очков заработанных клубом (победа - 3, ничья - 1)
    подсчет разницы забитых и пропущенных голов


Второй класс
Должен быть унаследован от первого. Расширить его методом, который подсчитывает общее количество игр для команды.


Алгоритм действий:

    создать классы
    в блоке if __name__ == '__main__': создать инстансы классов и продемонстрировать их работу
